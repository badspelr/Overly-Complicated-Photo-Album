User Manual

Introduction

Welcome to the Photo Album application! This manual will guide you through the main features and functionalities of the site.

Managing Albums

You can create, edit, and delete albums to organize your photos and videos.

  • Creating Albums: Navigate to "Create Album" from the main menu. Provide a title, description, and choose whether the album is public or private.
  • Uploading Media: You can upload photos and videos to any of your albums. Use the "Upload" page for bulk uploads or add media directly to a specific album.
  • Sharing Albums: You can create shareable links for your private albums to share them with specific people without making them public.

Custom Albums (Virtual Collections)

Custom Albums work like playlists for your photos and videos - they let you create virtual collections from media across different regular albums.

Understanding the Difference

  • Regular Albums: Like physical photo albums - each photo belongs to one album
  • Custom Albums: Like playlists - photos can appear in multiple custom albums

Creating and Using Custom Albums

  1. Create a custom album from your dashboard
  2. Edit any photo or video
  3. Select one or more custom albums from the dropdown
  4. Save - the photo now appears in those custom albums

Example Use Cases

  • "Best of 2024": Collect favorite photos from multiple albums
  • "Family Memories": Gather family photos from vacations, holidays, and birthdays
  • "Nature Photography": Build a collection of nature shots from various trips

Note: Photos stay in their original albums. Deleting from a custom album doesn't delete the original photo.

Keyboard Shortcuts

To help you navigate and use the application more efficiently, a number of keyboard shortcuts are available.

Global Navigation

These shortcuts work on any page.

  • Alt + D : Go to Dashboard
  • Alt + A : Go to My Albums
  • Alt + U : Go to Upload
  • Alt + S : Go to Search
  • Alt + F : Go to Favorites

Contextual Shortcuts

These shortcuts are available on specific pages.

  • Album Detail Page:
    • Ctrl + A : Select or deselect all photos.
    • Delete : Trigger the "Delete Selected" action for the selected photos.
  • Photo Detail Page:
    • ← (Left Arrow) : Navigate to the previous photo.
    • → (Right Arrow) : Navigate to the next photo.
    • F : Toggle the photo as a favorite.
  • Search Page:
    • / : Focus the main search bar.
